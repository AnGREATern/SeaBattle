# Продуктовое описание

Приложение представляет собой веб-игру "Морской бой", в которой 
пользователь может играть с ботом, предварительно выбрав сложность.

Игра делится на два основных этапа:

1. Расстановка кораблей - пользователь расставляет корабли на поле, может вращать и перемещать.
2. Непосредственно боевые действия. Игрок выбирает клетку, делает выстрел, получает результат. Далее получает результаты хода бота.

В конце игры победитель получает сообщение о победе или поражении.
