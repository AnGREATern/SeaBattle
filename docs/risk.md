# Вероятность

| Вероятность | Описание                                    |
| ----------- | ------------------------------------------- |
| Низкая      | событие маловероятно, вероятность менее 30% |
| Средняя     | событие может произойти, вероятность 30–70% |
| Высокая     | событие вероятно, вероятность более 70%     |

# Урон

| Урон             | Описание                                             |
| ---------------- | ---------------------------------------------------- |
| Средний          | влияет на отдельные задачи проекта, не нарушая сроки |
| Высокий          | вызывает значительные задержки                       |
| Катастрофический | приводит к потере автомата по предмету или отчислению|


# Воздействие

| Урон / Вероятность  | Высокая               | Средняя      | Низкая       |
| ------------------- | --------------------- | ------------ | ------------ |

| Высокий             | Высокое           | Высокое  | Среднее  |
